BuzzRef

A simple reference image viewer

A simple and powerful reference image viewer for digital artists and creatives. Organize your references with ease.

Download Now

Installation

Get the file for your operating system from the latest release.

Linux: Download the AppImage file, give it executable rights, then run it.

Windows: Download and run the executable file. No installation required.

macOS: Download and open the DMG file. You may need to allow the app in Security settings. See detailed instructions

Key Features

  • Image Management: Import images via drag & drop, paste from clipboard, or file browser. Move, scale, rotate, crop, and flip with ease.
  • Drawing & Sketching: Press 'D' to enter draw mode. Supports pressure-sensitive tablets with customizable brush size and color.
  • File Format Support: Native .bee format with embedded images. Import PureRef .pur files. Export to SVG or PNG.
  • Color Sampler: Pick colors from any image and copy hex values to your clipboard instantly.
  • Always On Top: Keep BuzzRef floating above other windows. Optional borderless mode for minimal distraction.
  • Keyboard Shortcuts: Fully customizable shortcuts for all operations. Display filename with 'F' key.
